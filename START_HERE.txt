┌──────────────────────────────────────────────────────────────────────────────┐
│  ⚠️  ВАЖНЫЕ ЗАМЕЧАНИЯ                                                        │
└──────────────────────────────────────────────────────────────────────────────┘

  • Django должен быть запущен на 0.0.0.0, НЕ на 127.0.0.1
  • В .env.production указывайте РЕАЛЬНЫЙ IP вашей машины
  • Устройства должны быть в ОДНОЙ WiFi сети
  • CORS_ALLOW_ALL_ORIGINS = True только для РАЗРАБОТКИ!
  • GitHub Pages = только статика, бэк нужно хостить отдельно


┌──────────────────────────────────────────────────────────────────────────────┐
│  🆘 TROUBLESHOOTING                                                          │
└──────────────────────────────────────────────────────────────────────────────┘

  "Connection refused"
  → Запустите: ./run_backend.sh (не python manage.py runserver!)

  "CORS error"
  → Проверьте application/settings.py: CORS_ALLOW_ALL_ORIGINS = True

  "Не могу подключиться по IP"
  → Проверьте WiFi сеть и firewall (порты 3000, 8000)

  Для полной диагностики:
  → Запустите: ./check_config.sh
